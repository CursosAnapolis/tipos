Informática Básica

Aprenda a utilizar ferramentas de informática essenciais para o mercado de trabalho atual.

Sobre o Curso

O curso de Informática Básica da Cursos Anápolis é voltado para mulheres que desejam adquirir conhecimentos fundamentais em informática, seja para uso pessoal ou profissional. Você aprenderá desde os conceitos básicos até o uso de ferramentas essenciais do pacote Office.

Conteúdo Programático

  • Conceitos básicos de hardware e software
  • Sistema operacional Windows
  • Internet e navegação segura
  • E-mail e comunicação digital
  • Editor de texto (Word)
  • Planilhas eletrônicas (Excel)
  • Apresentações (PowerPoint)
  • Noções de segurança digital

Metodologia

O curso é totalmente prático, com cada aluna utilizando um computador durante as aulas. As atividades são baseadas em situações reais do cotidiano profissional, garantindo que o aprendizado seja aplicável imediatamente.

Duração: 2 meses

Carga horária: 40 horas

Modalidade: Presencial

Benefícios

  • Acesso a computadores durante as aulas
  • Material didático incluso
  • Certificado reconhecido
  • Turmas reduzidas
  • Preparação para o mercado de trabalho
  • Orientação para criação de currículo
